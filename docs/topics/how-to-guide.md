# How-to guides

Use the plugin to publish Kotlin/Native code for .NET consumers, bind selected C# APIs for Kotlin,
or combine both workflows in one module. The guides use `MyCatLib`, a small library built around a
cat named Oreo.

- [Publish a Kotlin/Native library as NuGet](publish-kotlin-library-as-nuget.md)
- [Bind a NuGet package for Kotlin](bind-nuget-package-for-kotlin.md)

Check [Prerequisites](prerequisites.md) before starting. Publishing needs JDK 17 and Kotlin/Native.
Binding a NuGet package also needs the .NET 8 SDK or later.
