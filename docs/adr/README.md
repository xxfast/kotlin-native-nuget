- [030](030-property-exception-propagation.md) ‚Äî ADR-024 error-out-parameter extended to property getter/setter exports; both callable forms of nullable two-call pattern wrapped independently
- [031](031-constructor-exception-propagation.md) ‚Äî Constructor export returns `COpaquePointer?` (null on error) + `errorOut`; C# constructor checks error before assigning `_handle`; data class `copy()` receives same treatment
